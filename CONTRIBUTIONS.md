# Contributions

This repo is a backend / verification study built **on top of** a Tiny
Tapeout RTL submission by Robert Solomon Saab (Discord `.djharvey`).
The RTL itself is unchanged from his original submission; everything
that follows (physical flow runs, comparisons, formal verification,
tooling, write-ups) is by the maintainer of this fork.
